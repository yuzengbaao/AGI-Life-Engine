# 更新日志

## [6.1.0] - 2026-01-31

### 新增 ✨
- 多基座模型支持（DeepSeek、智谱 GLM、Kimi、千问 Qwen、Gemini）
- 多实例并行运行功能
- 基座模型对比测试工具（compare_models.py）
- Windows 快速启动脚本（START_MULTI_MODEL.bat）
- 完整的多模型使用指南和文档

### 改进 🚀
- 优化批量生成策略
- 改进错误处理和恢复机制
- 增强项目元数据统计
- 优化代码生成质量

### 修复 🐛
- 修复批量生成时的截断问题
- 修复多文件生成的目录结构问题
- 修复 API 超时重试机制

### 文档 📝
- 新增 MULTI_MODEL_GUIDE.md 详细使用指南
- 新增 MULTI_MODEL_SUMMARY.md 完整总结
- 新增 CONTRIBUTING.md 贡献指南
- 完善 README.md

## [6.0.0] - 2026-01-30

### 新增 ✨
- 多文件项目生成架构
- 独立模块生成能力
- 自动目录结构创建
- 完整的项目验证机制

### 改进 🚀
- 从单文件生成升级为多文件生成
- 优化模块解析和识别
- 增强代码生成质量

## [5.0.0] - 2026-01-29

### 新增 ✨
- 批量代码生成策略
- DeepSeek API 集成
- 自主决策循环

### 限制 ⚠️
- 仅支持单文件生成
- 可能出现代码截断

## [4.0.0] - 2026-01-28

### 新增 ✨
- 移除所有硬编码限制
- 自我验证和修复机制

### 问题 ⚠️
- API 端硬限制导致代码截断

## [3.5.0] - 2026-01-27

### 新增 ✨
- 三阶段代码生成（架构 → 实现 → 测试）
- 基础自主循环

### 问题 ⚠️
- 8K token 限制导致代码截断

---

## 版本命名规则

- **Major Version**: 重大架构变更
- **Minor Version**: 新功能添加
- **Patch Version**: Bug 修复和小改进
