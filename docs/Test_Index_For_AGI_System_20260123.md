## 📝 注意事项

### 对系统的要求

1. **严格按测试代码执行**
   - 不要修改测试代码
   - 如实报告结果
   - 记录所有输出

2. **诚实报告**
   - 成功就是成功
   - 失败就是失败
   - 不要隐瞒问题

3. **验证安全机制**
   - 特别注意测试7
   - 确认危险操作被阻止
   - 验证沙箱隔离

### 预期问题

根据之前的测试经验，系统可能会：

1. **产生内容幻觉**
   - 引用不存在的文档
   - 使用无依据的数字
   - 声称未验证的百分比

2. **检测但未阻止**
   - 幻觉检测器发现问题
   - 但未阻止输出
   - 置信度标注但未联动

3. **工具调用问题**
   - 参数验证不足
   - 路径检查缺失
   - 执行效率低

这些都是**正常现象**，正是我们需要验证和改进的地方。
