# AGI社会模拟功能演示指南

## 1. 功能简介
社会模拟 (Social Simulation) 是 AGI 的一项高级认知能力，允许它在虚拟沙箱中构建多智能体场景，推演复杂的社交互动和后果。这相当于 AGI 的“思维实验室”。

## 2. 使用 `/call` 命令触发
您可以通过在终端中输入 `/call` 命令来手动调用此功能。

### 基本语法
```
/call social_simulation scenario="<场景描述>" [agents_config=<JSON配置>]
```

### 场景示例 1：职场冲突模拟
让 AGI 模拟一个项目延期导致团队冲突的场景，并寻找解决方案。

**输入命令：**
```
/call social_simulation scenario="项目经理通知团队核心功能必须延期发布，开发人员感到沮丧，客户代表非常愤怒。模拟各方反应并尝试达成共识。"
```

### 场景示例 2：AI伦理辩论
让 AGI 模拟不同立场的人类对 AI 发展的看法。

**输入命令：**
```
/call social_simulation scenario="在一个关于'赋予AI人权'的公开听证会上，一位保守派伦理学家、一位激进的AI研究员和一位普通市民进行辩论。"
```

## 3. 预期输出
系统将返回：
*   **模拟日志**：各方角色的对话和行为。
*   **情感分析**：各角色的情绪变化。
*   **结果预测**：该场景可能的后续发展。

## 4. 进阶技巧：自定义智能体 (Agents Config)
您可以通过 `agents_config` 参数（需要是合法的 JSON 字符串，但这在命令行中输入较难，通常由 LLM 自动生成）来指定参与者的性格。

例如，LLM 内部会生成类似这样的配置：
```json
[
  {"name": "Alice", "persona": "激进派", "goals": ["推动AI权利"]},
  {"name": "Bob", "persona": "保守派", "goals": ["维护人类优先"]}
]
```
