
# 国家基本比例尺地图图式 第1部分：1:500 1:1000 1:2000 地形图图式 (GB/T 20257.1-2017)
# 核心要素摘要 (Knowledge Base)

## 1. 基本规定
- **坐标系统**: CGCS2000 (2000国家大地坐标系)
- **高程基准**: 1985国家高程基准
- **投影**: 高斯-克吕格投影 (3度带)
- **绘图单位**: 米 (m) 为基本单位 (1:1绘图)

## 2. 颜色标准 (RGB 参考)
- **黑 (Black)**: 居民地、人工设施、独立地物、负貌、图廓整饰 (0,0,0)
- **棕 (Brown)**: 地貌要素 (等高线、陡坎)、土质 (153,102,0)
- **蓝 (Blue)**: 水系要素 (0,0,255)
- **绿 (Green)**: 植被要素 (0,255,0)
- **红 (Red)**: 境界、控制点 (255,0,0)

## 3. 符号尺寸 (基于 1:500 比例尺换算)
*注: 图式规定的尺寸通常为打印后的毫米(mm)。在1:1(米)的CAD图中，需乘以比例尺分母/1000。*
*例: 1:500比例尺，1mm(图上) = 0.5m(实地)*

- **高程点 (Spot Height)**: 
  - 图式: 直径 φ1.0mm (实心或圈)
  - CAD尺寸: φ0.5m
  - 颜色: 黑 (图式规定) 或 棕 (部分行业习惯，以图式黑为准)
  - 注记: 2.0mm (CAD: 1.0m)

- **等高线 (Contour)**:
  - **首曲线 (Intermediate)**: 线宽 0.15mm (CAD: 0, 随层) -> 棕色
  - **计曲线 (Index)**: 线宽 0.30mm (CAD: 0.15m 模拟宽度或使用多段线宽度) -> 棕色
  - **注记**: 字高 2.0mm (CAD: 1.0m) -> 棕色, 位于线上或断开

- **控制点/界桩 (Control Points/Markers)**:
  - 图式: 外圈 φ3.0mm, 内圈 φ1.0mm
  - CAD尺寸: 外圈 φ1.5m
  - 颜色: 红 (控制点), 黑/褐 (一般界桩)

- **文字注记 (Annotations)**:
  - 汉字: 等线体/宋体, 字高 2.5mm~3.0mm (CAD: 1.25m ~ 1.5m)
  - 数字: 字高 2.0mm (CAD: 1.0m)

## 4. 图层命名 (参考 CASS/国标习惯)
- **KZD**: 控制点 (Red)
- **DGX**: 等高线 (Brown)
  - DGX-S: 首曲线
  - DGX-J: 计曲线
- **GCD**: 高程点 (Black/White)
- **JMD**: 居民地 (Black)
- **SXSS**: 水系设施 (Blue)
- **ZBZ**: 植被 (Green)
- **JJ**: 境界 (Red) - 保护区边界可归入此类
- **TK**: 图廓 (Black)

## 5. 图廓整饰
- 内图廓线: 0.1mm
- 外图廓线: 0.5mm (粗线)
- 必须标注: 
  - 密级 (可选)
  - 坐标系名称 "2000国家大地坐标系"
  - 高程基准 "1985国家高程基准"
  - 比例尺 "1:500"
  - 附注/测图时间/单位

## 6. 保护区特别规定 (用户自定义要素)
- 一级保护区: 红色实线/虚线，线宽 0.3mm
- 二级保护区: 黄色/橙色线
- 界桩: 需标注编号
